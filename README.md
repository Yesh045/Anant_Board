# AnantBoard

AnantBoard is a simple, web-based drawing application designed for quick sketching and note-taking. It features an intuitive interface with tools for drawing, erasing, and panning, along with dark mode support, infinite zoom, and the ability to export your creations as images. Built as a Progressive Web App (PWA), it can be installed on your device for offline use.

## Features

- **Drawing Tools**: Pen for freehand drawing with customizable color and thickness.
- **Eraser**: Erase parts of your drawing seamlessly.
- **Pan Mode**: Navigate around your canvas with hand tool.
- **Dark Mode**: Toggle between light and dark themes for comfortable viewing.
- **Infinite Zoom & Pan**: Zoom in and out infinitely and pan across the canvas.
- **Export**: Save your drawings as high-resolution PNG images.
- **Auto-Save**: Automatically saves your work to local storage.
- **PWA Support**: Install as a standalone app on mobile and desktop devices.
